皮亚杰《心理发生和科学史》读后感

—— D皮亚杰《心理发生和科学史》读后

  • 我偶尔翻一翻皮亚杰写的书，每一次都同样的心情：为他的结构主义哲学难以理解而烦恼，也为他的心理学发现而惊奇。
  • 记得多年以前，读《卡尔维诺文集》中随笔时，他特别提到过他的成长经历，说，他3-6岁的时候，母亲就为他准备了一年又一年的《美国邮报》漫画集，就等着他长大了可以读这些漫画。而《美国邮报》中的漫画是迪士尼式的，而不日本式的，对他的思维、想象的培养以及成人以后的文学创作极具积极意义。他是从漫画中学会幻想、从漫画的故事中学会哲学思考，而当时，他尚不能阅读。
  • 我对这断阅读经历记忆犹新，是因为我曾经在幼儿园兼职当过四年的园长，做了几年早期阅读的实验，也见识了自己孩子之外的别的小朋友。因此，当我读到《心理发生和科学史》第75页： > 因此，通过图形内关系的这种最初的几何结构 作用与历史连续的顺序是一致的， > 本书作都之一早就发现，儿童设想的最初空间形式与历史相反，与理论顺序致，具有拓扑性质，而欧里德和投影几何图形是在较晚出现的。……
  • 这段话，可以理解3至6岁的小朋友喜欢拓扑漫画（迪士尼式），而不是喜欢欧几里德漫画（日本式）。皮亚杰的心理发生过程与实际课程协调一致，可以加强和促进思维的发展。
  • 课程设计从皮亚杰的观点看来，就得课程内容、课程的形式得与心理发生过程相一致，这样增进成长与发育。
  • 这里提到一个反方向的顺序，如何理解呢？
  • 按照数学史的发展顺序：欧几里德几何-非欧几里德几何-拓扑几何这个顺序发展，而心理发生过程却在欧几里德之前，出现了对拓扑几何敏感与兴趣，因此我们可以将漫画教育放在幼儿园阶段，从漫画中讲故事，发展小朋友的口头表达能过，从漫中编故事，发展小朋友的想象能力等。、

Installation

Dillinger requires Node.js v4+ to run.

You need Gulp installed globally:

$ npm i -g gulp

$ git clone [git-repo-url] dillinger
$ cd dillinger
$ npm i -d
$ NODE_ENV=production node app

Docker

Dillinger is very easy to install and deploy in a Docker container.

By default, the Docker will expose port 80, so change this within the Dockerfile if necessary. When ready, simply use the Dockerfile to build the image.

cd dillinger
docker build -t <youruser>/dillinger:latest .

This will create the dillinger image and pull in the necessary dependencies. Once done, run the Docker and map the port to whatever you wish on your host. In this example, we simply map port 80 of the host to port 80 of the Docker (or whatever port was exposed in the Dockerfile):

docker run -d -p 80:80 --restart="always" <youruser>/dillinger:latest

Verify the deployment by navigating to your server address in your preferred browser.
